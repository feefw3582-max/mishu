PWA / Server Push

iPhone准点提醒必须接服务端

你实测是对的：iPhone 13 会杀后台，纯前端定时器不可靠。这个页面现在只负责安装PWA、申请通知权限、注册Push订阅；真正按点推送必须由服务端定时触发。

安装步骤
  1. iPhone 用 Safari 打开线上地址。
  2. 点底部分享按钮，选择“添加到主屏幕”。
  3. 从主屏幕打开“Raymond待办”。
  4. 点“测试通知权限”，确认系统通知能弹出。
  5. 部署服务端后，点“订阅服务端推送”。
